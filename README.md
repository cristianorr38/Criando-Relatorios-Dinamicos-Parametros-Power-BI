## 📌 Sobre o Projeto
Este projeto faz parte do **Desafio de Projeto - Criando Relatórios Dinâmicos com o uso de Parâmetros no Power BI**.  
O objetivo foi **criar relatórios interativos e dinâmicos**, aplicando parâmetros para diferentes visões dos dados, com foco em storytelling e experiência do usuário.

As principais diretrizes seguidas foram:
- Criação de parâmetros com base em **categorias** (ex.: país, produto, segmento).
- Criação de parâmetros com base em **valores** (ex.: profit, sales, units sold).
- Manutenção da estilização consistente do relatório.
- Construção de uma narrativa visual para apresentação ao cliente.
